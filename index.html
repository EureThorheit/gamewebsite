Spielesammlung

Ich habe eine Website mit drei Spielen erstellt.

  1. Tetris
  2. Reversi
  3. Conways Game of Life
Ich habe alle Spiele in Rust geschrieben und mit wasm (Web Assembly) in den Browser gebracht. Aber ich habe nicht nur wasm verwendet, sondern auch Javascript für das Zeichnen auf einem Canvaselement und dem Verarbeiten von Tastatureingaben. Durch das Verwenden von wasm kann man die Seiten nicht mehr im Browser direkt laden.
Am einfachsten ist es, einen Server zu erstellen, der wasm ausführen kann. Es gibt verschiedene Möglichkeiten einen Server zu erstellen doch ein python local host server ist am einfachsten.
Mit dem python befehl:

python -m http.server 8080

kann man einen local host server erstellen.
Oder man nutzt miniweb.exe mit dem localhost: http://localhost:8000 und startet die exe mit dem Befehl:

miniweb.exe -p 8000 -r .

In Tetris muss man die fallenden Blöcke in einer Reihe anordnen, dass sie ausgefüllt wird. Wenn eine Reihe gefüllt ist, leert sie sich und man bekommt Punkte. Zum bewegen der fallenden Blöcke nutzt man: WASD oder Pfeiltasten und zum Rotieren nutzt man: QE

Reversi hat ein 8x8 Felder Spielbrett. Es gibt zwei Farben Rot und Grün. Die Spieler setzen nacheinander auf das Spielbrett, sodass mindestens ein Stein der gegnerischen Farbe zwischen seinen eigenen Steinen liegt. Die steine die beim Zug zwischen den eigenen Steinen und dem neugesetzten stein liegt werden umgedreht. Hier eine ausführliche Anleitung

Game of Life ist eine Simulation oder ein Nullspieler spiel, welches auf einfachen Regeln basiert aber tortzdem komplexes verhalten entsteht. Die Welt ist in Zellen aufgeteilt. Eine Zelle kann entweder aktiv oder inaktiv sein. Eine Zelle wird in der nächsten Iteration aktiv wenn sie genau drei Nachbarn hat. Und sie bleibt am leben Wenn sie 2 oder 3 Nachbarn hat.
